# 🚗 Auton liikkuminen

This is a Python GUI application built with Tkinter that simulates car movement.

## 🔧 Features

- Fuel consumption system
- Tire condition (can break randomly)
- Car movement on a map (Canvas)
- Speed control using a slider (Scale)
- Ability to refuel and change tires
- Visual feedback (color changes, messages)

## 🖥️ How it works

The user selects a route and starts the car.  
The car moves step by step while:
- fuel decreases
- tires may break
- the car stops if there is a problem

The simulation can continue after fixing issues.

## ▶️ How to run

Make sure you have Python installed.

Run the program:

```bash
python main.py
```
